# 🛡️ Botium Toys Internal Security Audit

## Overview

This project presents a comprehensive internal security audit for **Botium Toys**, a fictional retail organization used as a case study in the **Google Cybersecurity Professional Certificate – Course 2: Play It Safe: Manage Security Risks**.

The audit evaluates the organization's security posture by reviewing existing controls, identifying risks, assessing compliance requirements, and recommending practical improvements. The project demonstrates a structured security audit process similar to those performed by entry-level cybersecurity analysts.

---

## Project Objectives

* Define the scope and objectives of the security audit.
* Evaluate administrative, technical, and physical security controls.
* Assess compliance with applicable security standards.
* Identify and prioritize organizational risks.
* Develop a practical security improvement plan.
